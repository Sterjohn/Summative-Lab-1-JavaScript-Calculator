# JavaScript Calculator

A simple calculator that can add, subtract, multiply, and divide two numbers. It also keeps track of all calculations in a history log.

## How to Run

1. Download the calculator.js file
2. Open your terminal
3. Type: node calculator.js

## What it Does

- Adds two numbers
- Subtracts two numbers
- Multiplies two numbers
- Divides two numbers
- Saves each calculation to a history array
- Displays the full history of calculations
